# LLS Resource Monitor

Simple resource monitor written in ZSH to track live CPU and RAM usage. 

**CLONE THE REPO:
```zsh
git clone https://github.com/shoobham84/lls-monitor
cd lls-monitor
chmod +x lls_monitor.sh
```

**TO RUN:
```zsh
./lls_monitor.sh
```

**TO RUN SILENTLY IN BACKGROUND:
```zsh
nohup ./lls_monitor.sh > lls_monitor.log 2>&1 &
```
